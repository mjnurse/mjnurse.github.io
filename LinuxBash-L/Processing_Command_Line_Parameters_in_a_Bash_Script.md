---
title: Processing Command Line Parameters in a Bash Script
---

Option 1:

Process a single string of parameters which start with a `-`.

```bash
param=$1

# Only if parameters start with a '-'

while [[ "${param:0:1}" == "-" && "$param" != "" ]]; do
  case $param in
    *a*)
      echo a
      param="${param//a/}"
      ;;
    *b*)
      echo b 
      param="${param//b/}"
      ;;
    *)
      param=""
      ;;
  esac
done
```
