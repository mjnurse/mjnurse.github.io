## 2. Install Apache, create a default web page.

```bash
> sudo apt install apache2
mjnlearnsgcp@mjn-web-server:~$ sudo su
root@mjn-web-server:/home/mjnlearnsgcp# echo "hostnamectl | grep hostname > /var/www/html/index.html" > /onboot.sh
root@mjn-web-server:/home/mjnlearnsgcp# chmod +x /onboot.sh 
root@mjn-web-server:/home/mjnlearnsgcp# 

echo "@reboot /onboot.sh" > mycron
root@mjn-web-server:/home/mjnlearnsgcp# crontab mycron


3. Create an image of the boot disk.
4. Create an instance template and use the boot disk image (a custom image).
5. Create an instance group using the above instance template.
6. Create a load balancer with the instance group as the backend.
