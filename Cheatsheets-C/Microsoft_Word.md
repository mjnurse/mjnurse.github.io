## Shortcut Keys

|-----------|-----------------------------------|
| ALT-1 ..9 | Quick Action Toolbar Command      |

## Word Search Replace

| Characters | Match Description          |
|------------|----------------------------|
| `^l`       | Manual line break          |
| `^p`       | Paragraph break            |
| `^n`       | Column break               |
| `^m`       | Manual page break          |
| `^b`       | Section break              |
| `^t`       | Tab character              |
| `^w`       | White space (space or tab) |
| `^s`       | Nonbreaking space          |
| `^~`       | Nonbreaking hyphen         |
| `^-`       | Optional hyphen            |
| `^=`       | En dash                    |
| `^+`       | Em dash                    |
| `^^`       | Caret (^)                  |
| `^%`       | Section symbol             |
| `^v`       | Paragraph symbol           |
| `^?`       | Any character              |
| `^$`       | Any letter                 |
| `^#`       | Any digit                  |
| `^e`       | Endnote mark               |
| `^g`       | Graphic                    |
