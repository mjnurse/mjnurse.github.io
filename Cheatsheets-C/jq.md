## Multi-level filter

```json
{
  "a": 1,
  "b": {
    "c": 2,
    "d": 3
  },
  "e": [
    {
      "f": 6,
      "g": 7
    },
    {
      "f": 8,
      "g": 9
    }
  ]
}

```

Command: `jq '.b |= {c} | .e[] |={g} | {e, b, a}'``

```json
{
  "a": 1,
  "b": {
    "c": 2
  },
  "e": [
    {
      "g": 7
    },
    {
      "g": 9
    }
  ]
}
 ```
