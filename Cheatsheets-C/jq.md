## Multi-level filter

```json
{
  "a": 1,
  "b": {
    "c": 2,
    "d": 3
  },
  "e": {
    "f": 6,
    "g": 7
  }
}
```

Command: `jq '.b |= {c} | .e |={g}'``

```json
{
  "a": 1,
  "b": {
    "c": 2
  },
  "e": {
    "g": 7
  }
}
 ```
