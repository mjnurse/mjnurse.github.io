### Cloud Compute
----
#### Step 1: Authenticating With Google Cloud

##### Create a service account to connect with

- In the GCP Console, go to the IAM & admin > Service accounts page and click CREATE SERVICE ACCOUNT.

- Enter account name - I used: "cli_connect".  Click CREATE.

- From the Role list, select Project > Owner.  Click CONTINUE.

- Click CREATE KEY.  Choose JSON.  Click CREATE.

- A JSON file that contains your key downloads to your computer.

- Click DONE.

##### Create environment variable to point to the JSON credentials file

Copy the credentials file to home directory in a .credentials directory.
```
> mkdir .credentials
> mv /mnt/c/Users/Martin/Downloads/mjnurseuk-website-fc2433bc8905.json .credentials/mjnurseuk-website-cli-connect.json
```
Add `export GOOGLE_APPLICATION_CREDENTIALS="/home/martin/.my-credentials/mjnurseuk-website-cli-connect.json"` to the `.bashrc` file.
