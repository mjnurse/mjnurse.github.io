At a high level, Elasticsearch has 3 caches:

- Page cache
- Shard-level request cache
- Query Cache

# Page Cache

The OS managed disk cache (sometimes call the file cache).  Caches blocks of data read from disk and can give a big increase in performance where there is lots of re-use.
